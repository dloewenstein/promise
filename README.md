Installation
------------

You can install promise from github with:

``` r
# install.packages("devtools")
devtools::install_github("dloewenstein/promise")
```

You can install promise from Anaconda.org with:

``` shell
conda install -c dloewenstein r-promise
```

Example
-------

In R

``` r
library("promise")
data(promise)
```
